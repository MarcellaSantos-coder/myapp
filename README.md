# Catálogo Interativo Mobile

Aplicativo mobile desenvolvido com React Native e Expo para exibir produtos de uma loja online organizados por categorias.

## Tecnologias utilizadas

- React Native
- Expo
- Axios
- React Navigation

## Funcionalidades

- Tela de Login com validação
- Listagem de produtos por categoria
- Consumo de API externa
- Tela de detalhes do produto
- Logout do usuário

## API utilizada

https://dummyjson.com/docs

## Como executar o projeto

1. Instalar dependências

npm install

2. Rodar o projeto

npx expo start
